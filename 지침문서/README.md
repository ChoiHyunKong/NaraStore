# 나라장터 크롤링 및 제안서 분석 서비스

## 개요
이 서비스는 나라장터(G2B)의 입찰공고를 키워드로 검색하여 크롤링하고, 제안요청서(RFP) 파일을 AI로 분석하여 수주 전략을 수립해주는 도구입니다.

## 설치 및 실행 방법

1. **가상환경 생성 및 패키지 설치** (최초 1회)
   ```bash
   python -m venv venv
   .\venv\Scripts\pip install -r requirements.txt
   ```

2. **서비스 실행**
   ```bash
   .\venv\Scripts\streamlit run app.py
   ```

## 주요 기능

### 1. 입찰공고 검색/크롤링
- 키워드(예: "인공지능", "빅데이터")를 입력하면 나라장터에서 검색합니다.
- 검색 결과(공고명, 공고번호, 기관, 링크 등)를 화면에 표시하고 **CSV 파일**로 다운로드할 수 있습니다.
- **주의**: 크롤링 시 Chrome 브라우저가 자동으로 실행되었다가 종료됩니다. (Selenium 사용)

### 2. 제안서 AI 분석
- 제안요청서 파일(HWP, PDF, PPTX)을 업로드합니다.
- **OpenAI API Key**를 입력하고 분석을 요청합니다.
- AI가 다음 항목을 분석하여 리포트를 생성합니다:
  - 클라이언트 니즈
  - 기능 요구사항
  - 제약 사항
  - 필수 사항
  - 수주 전략

## 주의사항
- **HWP 파일**: 한글 파일의 구조적 복잡성으로 인해 텍스트 추출이 완벽하지 않을 수 있습니다. (표나 특수 서식 등)
- **API 비용**: AI 분석 기능은 OpenAI API를 사용하므로 개인의 API Key가 필요하며 비용이 발생할 수 있습니다.
- **법적 고지**: 크롤링한 데이터는 개인적인 용도로만 사용해야 하며, 상업적으로 재배포 시 법적 문제가 발생할 수 있습니다.
